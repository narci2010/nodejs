#### README

- 作用：自动生成1-n个.js文件名的 js文件

- app文件放入全局node中的 node_modules 目录下
- app.cmd 放入全局node的目录下

```
v6.9.2              一级目录
------node_modules  二级目录
----------app       三级目录
----------npm       三级目录
------node.exe      二级目录
------app.cmd       二级目录
------npm.cmd       二级目录
```

- cmd 中 输入:app 10  表示创建1.js -- 10.js
